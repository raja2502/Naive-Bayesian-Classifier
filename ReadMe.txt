1.The code is written in python
2.You need have python-weka-wrapper3 to run this project 
3.You can install python-weka-wrapper3 using pip
4.LIAC-ARFF is used to import the .arff file in the 3rd module of the code
5.LIAC-ARFF can be downloded from pip
6.The code assumes that the .arff files are placed in the same folder as the python code.
7.Model is saved to the same folder in which the code is placed.
8.For the first case.arff for the file name should not be given
9.In the second case filenames have to be given with extension
